## 1. 简介

这是一个给抖m们设计的角色扮演小游戏。

在这个游戏中，你可以选择自己的角色，完成角色的任务，拯救这个国家。

它的玩法类似于faproulette.co上的轮盘游戏，但是它更加程序化的提供了轮盘的过程。
